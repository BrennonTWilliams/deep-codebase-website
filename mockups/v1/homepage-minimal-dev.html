core features

Intelligent Code Selection

Hybrid TF-IDF semantic analysis with call graph dependency tracking. Automatically identifies relevant code snippets by relevance and architectural significance.

Smart Query Understanding

Intent detection adapts results based on your query. Ask for architecture, integration, or extraction—get optimized results for each scenario.

Complete Dependency Mapping

Automatic dependency tracing through static analysis. Extract everything needed—no missing imports, no broken references.

Polyglot Code Intelligence

Fast multi-language parsing with deep React/TSX support. Understands components, hooks, lifecycle methods, and JSX patterns.

Contextual Function Naming

Intelligent naming using variable assignments and callback context. Self-documenting snippet names for instant code identification.

Integration-Ready Output

Comprehensive markdown reports with usage examples, integration notes, and architectural context. Line references and call flows clearly documented.

Python 3.8+
1

Install via pip

pip install deep-codebase
2

Basic analysis (AI only)

deep-codebase analyze <repository-url> "How does authentication work?"
3

Extract with code snippets (recommended)

deep-codebase analyze <repository-url> "Extract auth system" --include-snippets 12
